# 离线图片水印工具：保护隐私的免费在线工具

> 在线体验地址：[https://watermark.cdtools.click](https://watermark.cdtools.click)

![kMUfvy](https://cdn.jsdelivr.net/gh/houxiaozhao/imageLibrary@master/uPic/2025/02/10/kMUfvy.png)

## 背景介绍

在日常工作和生活中，我们经常需要给图片添加水印，比如：

- 证件复印件需要标注用途
- 文档资料需要防泄露标记
- 照片需要版权保护
- 企业内部资料需要保密标记

市面上虽然有很多水印工具，但大多存在以下问题：

1. 需要上传图片到服务器，存在隐私泄露风险
2. 收费或有使用次数限制
3. 需要下载安装专门的软件
4. 操作复杂，不够便捷

为了解决这些问题，我开发了这款完全免费的离线图片水印工具。它完全在浏览器本地运行，无需上传图片，保护用户隐私安全。

## 核心功能

### 1. 完全离线处理

- 所有图片处理都在浏览器本地完成
- 无需上传到服务器
- 保护隐私和敏感信息安全

### 2. 便捷的水印添加

- 支持拖拽上传图片
- 支持粘贴上传图片
- 支持批量处理多张图片
- 实时预览水印效果

### 3. 丰富的水印选项

- 自定义水印文字
- 内置多种常用模板：
  - 办事模板：仅供办理 XX 使用，他用无效
  - 认证模板：仅用于 XX 认证，他用无效
  - 复印核验：复印件与原件相符
  - 证件模板：该证件仅供 XX 查看，不得他用
  - 内部资料：内部资料，请勿外传
  - 版权声明：版权所有，严禁外传

### 4. 灵活的布局方式

- 平铺模式：水印均匀分布在整个图片上
- 单个水印：可选择水印位置（居中、左上角、右上角、左下角、右下角）
- 自定义数量：可指定水印的数量和分布

### 5. 专业的样式设置

- 字体选择：支持多种中英文字体
- 字体样式：支持粗体、斜体
- 颜色设置：自定义水印颜色
- 透明度调节：调整水印透明度
- 大小控制：调整水印字号
- 间距设置：调整水印之间的间距
- 旋转角度：自定义水印的倾斜角度

### 6. 便捷的操作功能

- 一键下载所有处理后的图片
- 支持复制单张图片到剪贴板
- 支持删除单张或全部图片
- 支持预览和放大查看
- 支持键盘快捷键操作

## 使用说明

### 基本操作流程

1. 上传图片
   - 点击上传区域选择图片
   - 直接拖拽图片到页面
   - 复制图片后直接粘贴
2. 设置水印
   - 输入水印文字或选择模板
   - 选择布局方式
   - 调整样式参数
3. 导出图片
   - 点击"下载全部"打包下载
   - 点击单张图片的"下载"按钮
   - 点击"复制"按钮复制到剪贴板

### 键盘快捷键

- `Ctrl/Command + D`：下载全部图片
- `Ctrl/Command + V`：粘贴图片
- `Delete`：删除全部图片
- `Ctrl/Command + Z`：撤销删除
- `Esc`：关闭预览
- `←/→`：预览模式下切换图片
- `1-9`：快速调整水印大小
